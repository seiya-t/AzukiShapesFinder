AzukiShapesFinder
=================
Microsoft Excelの図形(旧オートシェイプ)検索アドインです。  
Excelの基本機能では図形内のテキスト検索ができないので、作りました。  
Excel2010、2013にて動作を確認しています。  
そのほかのバージョンでは、動作を確認していません。  
動作に関する報告をいただければ幸いです。  

使い方
-----
### インストール方法 ###
Microsoft社のWebページ( http://office.microsoft.com/ja-jp/excel-help/HP005203732.aspx )をご覧ください。

### 起動方法 ###
アドインを有効にした状態で、キーボードから [Ctrl]+[l] を入力すると、フォームが開きます。

### 操作方法 ###
1. 検索キーワードを入力します。  
2. 検索ボタンをクリックします。  
3. 検索結果一覧に条件にあてはまる図形内のテキストの一覧が表示されるので、そのテキストを選択すると、そのテキストが含まれる図形を選択状態にします。  

### アンインストール方法 ###
Microsoft社のWebページ( http://office.microsoft.com/ja-jp/excel-help/HP005203732.aspx )をご覧ください。

問題
-----
現在、以下の問題がある事が分かっています。  
Excel VBAに詳しい方、協力をお願いします。  

### 図形の線など、テキストを入力できない図形が存在すると、エラーになる ###
図形の線など、TextFrameプロパティが存在しない図形がシート上に存在すると、エラーになります。  
現在はエラーの発生を無視するようにしています。  
そのため、アドインを使用する上で困ることはありません。  

### 図形の線の中にある曲線を使用すると、テキストが入力できるにも関わらず検索できない ###
曲線は、タイプがDrawingです。  
Drawingはテキストを入力できるにもかかわらずTextFrameを持たないようです。  
そのため、上の対応に伴ってエラーとなることはありませんが、正常に検索できません。  
なお、公開されている他のフリーの図形検索アドインもいくつか試してみましたが、同様の現象が発生しているため、同じような回避方法を行っていると思われます。
